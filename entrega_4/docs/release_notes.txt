# Instruções de como executar o jogo Conjunto
-- Versão do Python: 3.11.1

Primeiramente, crie um ambiente virtual na subpasta `venv`:

```sh
python -m venv venv
```

Em seguida, entre no venv:

```sh
source venv/bin/activate
```

Instale os pacotes necessários:

```
pip install -r requirements.txt
```

Para executar o jogo, entre no diretório `src` e rode o script:

```sh
cd src
python3 main.py
```
