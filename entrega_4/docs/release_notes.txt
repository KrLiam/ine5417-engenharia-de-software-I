# Instruções de como executar o jogo Conjunto
-- Versão do Python: 3.11.1

Primeiramente, abra um terminal no diretório principal do projeto (o diretório
contendo as pastas `diagramas`, `docs`, `src` etc.

Crie um ambiente virtual na subpasta `venv`:

```sh
python -m venv venv
```

Em seguida, entre no venv. No Linux, isso pode ser feito da seguinte forma:

```sh
source venv/bin/activate
```

Instale os pacotes necessários:

```
pip install -r requirements.txt
```

Para executar o jogo, entre no diretório `src` e rode o script:

```sh
cd src
python3 main.py
```
